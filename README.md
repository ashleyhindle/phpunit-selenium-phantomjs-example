# phpunit-selenium-phantomjs-example
PhpUnit Selenium PhantomJS Integration Testing

Required PHP 5.6 - earlier versions of PHPUnit and phpunit-selenium aren't terribly good

# How to use

* `vagrant up`
* `./test.sh`
* New tab: `composer test`
* Success!
